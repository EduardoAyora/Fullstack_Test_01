# Backend - API REST

Este directorio contendrá tu implementación del backend de la aplicación.

## Stack Requerido

- **Runtime**: Node.js v18+
- **Framework**: Express.js
- **Lenguaje**: TypeScript
- **Base de Datos**: MySQL o MongoDB
- **Autenticación**: JWT
- **Documentación**: Swagger/OpenAPI

## Librerías Sugeridas (Opcionales)

Puedes usar las librerías que consideres apropiadas. Algunas opciones comunes:

### Base de Datos
- **MySQL**: `mysql2`, `sequelize`, `sequelize-typescript`, `typeorm`
- **MongoDB**: `mongoose`

### Validación
- `express-validator`, `joi`, `zod`, `yup`

### Documentación
- `swagger-jsdoc`, `swagger-ui-express`

### Testing
- `jest`, `mocha`, `supertest`, `chai`

### Seguridad
- `bcrypt`, `helmet`, `cors`, `express-rate-limit`

### Utilidades
- `dotenv`, `jsonwebtoken`
